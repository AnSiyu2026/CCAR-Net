# CCAR-Net
Official repository of Constraining impact flux of Late Heavy Bombardment through intelligent predicting the lunar-wide crater ages
## Predict 
The code in predict.py uses the trained CCAR-Net weights to perform age regression prediction on craters in the Robbins crater database.
The weight file for CCAR-Net is saved as checkpoint.pt and can be downloaded from the following link: https://doi.org/10.5281/zenodo.19651328
Robbins crater database can be obtain via: https://astrogeology.usgs.gov/search/map/Moon/Research/Craters/lunar_crater_database_robbins_2018.xml. Thank them for their excellent work!
## Result
The file crater_ages_with_uncertainty.csv stores the age regression results generated by CCAR-Net for craters with diameters greater than 8 km in the Robbins crater database, including the upper and lower bounds of 5% uncertainty.
